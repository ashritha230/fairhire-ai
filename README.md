# FairHire AI – Resume Screening & Ranking System

FairHire AI is a web-based intelligent recruitment assistant that automates the resume screening process. It helps recruiters quickly identify the most relevant candidates by analyzing resumes and ranking them based on their similarity to the job description.

## Features
- Upload multiple candidate resumes
- Input job description and requirements
- NLP-based resume parsing
- AI-powered resume ranking
- Interactive Streamlit dashboard
- Automated candidate scoring

## Technologies Used
- Python
- Streamlit
- spaCy
- Sentence Transformers
- Scikit-learn
- Pandas

## How It Works
1. Recruiter uploads resumes.
2. The system extracts text and relevant skills.
3. The job description is analyzed.
4. AI models compute similarity scores.
5. Candidates are ranked from best to least match.

## Project Purpose
This project demonstrates how Artificial Intelligence and Natural Language Processing can automate recruitment workflows and improve hiring efficiency.
